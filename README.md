# 🛍️ Product Listing App

A full-stack web application that displays products with features like search, category filtering, and price sorting. Built as part of an internship assignment.

---

## 🚀 Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Data:** Mock API / Static JSON

---

## ✨ Features

* 📦 Display products (name, price, category)
* 🔍 Search products by name
* 🏷️ Filter by category
* 💰 Sort by price (Low → High / High → Low)
* 📱 Responsive UI using Tailwind CSS

---

## ⚙️ API Endpoints

### `GET /products`

Query Parameters:

* `search` → filter by product name
* `category` → filter by category
* `sort` → `asc` | `desc` (price sorting)
